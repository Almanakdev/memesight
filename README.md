# MemeSight Demo App

A demo landing app for **MemeSight AI** — a multi-agent intelligence platform for the pump.fun ecosystem on Solana.

## Stack

- React
- Vite
- Lucide React icons

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project structure

```text
memesight-demo/
├── index.html
├── package.json
├── favicon.svg
└── src/
    ├── App.jsx
    ├── main.jsx
    └── styles.css
```

## Notes

This is a front-end starter meant for GitHub upload and quick iteration.
You can expand it with:

- real pump.fun / Solana data APIs
- wallet graph visualizations
- token detail pages
